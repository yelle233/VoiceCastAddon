# 语音施法附属 - 快速上手指南

## 简介

为 Iron's Spells 'n Spellbooks 模组添加语音识别功能，通过说话来释放法术。

## 前置要求

- Minecraft 1.21.1 + NeoForge
- Iron's Spells 'n Spellbooks 模组

## 安装步骤

### 1. 安装模组

将 `voicecastaddon-x.x.x.jar` 放入 `mods` 文件夹。

**注意：** 模组已内置中英文语音识别模型，无需额外下载！

### 2. 生成法术配置

首次进入游戏后，运行命令生成法术配置：

```
/voicecast generate
```

这会自动扫描所有已安装的法术并生成配置文件。

### 3. 绑定按键

在游戏设置中找到"语音施法附属"分类，绑定"按住以语音施法"按键。

## 开始使用

1. 按住**语音施法键**
2. 对着麦克风说出法术名称（如"火球"、"治疗术"）
3. 松开按键，系统识别后自动释放法术

## 自定义触发词

编辑配置文件：`config/voicecastaddon/spell_aliases.json`

```json
{
  "irons_spellbooks:fireball": [
    "fireball",
    "火球",
    "火球术"  // 可以添加多个别名
  ]
}
```

修改后运行 `/voicecast reload` 热重载配置。

## 常用命令

- `/voicecast list` - 查看所有已注册的法术
- `/voicecast generate` - 自动生成/更新法术配置
- `/voicecast reload` - 重新加载配置文件

## 语音识别技巧

1. **说话清晰**：语速适中，发音清楚
2. **使用简短词汇**：推荐2-4个字的触发词（如"火球"、"治疗"）
3. **避免复杂词汇**：不要使用生僻字或长句子
4. **减少噪音**：在安静环境下使用
5. **避免重复别名**：不同法术不要使用相同的触发词

## 设备设置

游戏内按 `ESC` → `Mod选项` → `语音施法附属` 可以选择麦克风设备。

## 常见问题

**Q: 识别不准确怎么办？**
A:
- 使用更简短、常见的触发词（推荐2-4个字）
- 说话清晰，语速适中
- 在安静环境下使用
- 使用质量更好的麦克风
- 如需更高准确度，可下载更大的自定义模型（见下方"高级功能"）

**Q: 提示"未拥有此法术"？**
A: 检查配置文件中是否有重复的别名，确保不同法术的触发词唯一。

**Q: 首次按键有延迟？**
A: 首次加载语音模型需要时间（约1-2秒），后续使用会流畅。

**Q: 如何选择麦克风？**
A: 游戏内按 `ESC` → `Mod选项` → `语音施法附属` 可以选择输入设备。

**Q: 如何跳过法术读条？**
A: 在配置文件中设置 `"skipCastTime": true`，语音施法将立即释放法术。这样说话时间可以弥补读条时间，适合快速战斗。

**Q: 如何无视法术冷却？**
A: 在配置文件中设置 `"ignoreCooldown": true`，语音施法将无视冷却时间。注意这仅影响语音施法，正常施法仍受冷却限制。

## 高级功能

### 双模式识别：离线 + 在线

模组支持三种识别模式和四种在线提供商：

#### 1. 离线模式（默认）
- 使用内置 VOSK 模型
- 完全离线，无需网络
- 隐私安全，响应快速
- 适合简短触发词（2-4个字）

#### 2. 在线模式
- 支持多个语音识别服务提供商
- 需要网络连接和 API 密钥
- 识别准确度高，支持长句子、诗句、网络梗词
- 适合复杂触发词

**支持的提供商：**
- **腾讯云**：免费额度 15,000次/月
- **百度AI**：免费额度 50,000次/天（推荐）
- **阿里云**：免费额度 2小时/天
- **讯飞**：免费额度 500次/天

#### 3. 混合模式（推荐）
- 优先使用在线识别，失败时自动回退到离线模式
- 兼顾准确度和可用性

### 配置在线识别

编辑配置文件：`config/voicecastaddon/client_settings.json`

#### 使用腾讯云
```json
{
  "recognitionMode": "hybrid",
  "onlineProvider": "tencent",
  "tencentSecretId": "你的SecretId",
  "tencentSecretKey": "你的SecretKey"
}
```

#### 使用百度AI（推荐，免费额度最高）
```json
{
  "recognitionMode": "hybrid",
  "onlineProvider": "baidu",
  "baiduApiKey": "你的API_Key",
  "baiduSecretKey": "你的Secret_Key"
}
```

#### 使用阿里云
```json
{
  "recognitionMode": "hybrid",
  "onlineProvider": "aliyun",
  "aliyunAccessKeyId": "你的AccessKeyId",
  "aliyunAccessKeySecret": "你的AccessKeySecret",
  "aliyunAppKey": "你的AppKey"
}
```

#### 使用讯飞
```json
{
  "recognitionMode": "hybrid",
  "onlineProvider": "xfyun",
  "xfyunAppId": "你的APPID",
  "xfyunApiKey": "你的APIKey",
  "xfyunApiSecret": "你的APISecret"
}
```

**配置说明：**
- `recognitionMode`: 识别模式
  - `"offline"` - 离线模式（默认）
  - `"online"` - 在线模式
  - `"hybrid"` - 混合模式（推荐）
- `onlineProvider`: 在线提供商（`tencent`、`baidu`、`aliyun`、`xfyun`）
- `skipCastTime`: 跳过法术读条（`true` 或 `false`，默认 `false`）
  - 开启后，语音施法将立即释放法术，无需读条
  - 说话时间可以弥补读条时间
  - 适合需要快速反应的战斗场景
- `ignoreCooldown`: 无视法术冷却（`true` 或 `false`，默认 `false`）
  - 开启后，语音施法将无视冷却时间限制
  - 冷却期间也可以释放法术
  - 注意：仅影响语音施法，正常施法仍受冷却限制
- `longSentenceThreshold`: 长句子阈值（混合模式下，超过此字数使用在线识别）

### 使用自定义语音模型（可选）

模组已内置小型中英文模型（vosk-model-small-cn-0.22 和 vosk-model-small-en-us-0.15），可满足日常使用需求。

**重要提示：** 由于 VOSK 库的限制，某些大型模型（如 vosk-model-cn-0.22、vosk-model-en-us-0.22）可能因内存不足或格式不兼容而无法加载。如果遇到加载失败，模组会自动回退到内置的小型模型。

#### 如何使用自定义模型：
1. 从 https://alphacephei.com/vosk/models 下载模型
2. 解压模型文件
3. 将解压后的文件夹放入：`config/voicecastaddon/custom-voice-models/`
4. 确保目录结构正确：
   ```
   config/voicecastaddon/custom-voice-models/
   └── vosk-model-small-cn-0.22/
       ├── am/
       │   └── final.mdl
       ├── conf/
       │   ├── mfcc.conf
       │   └── model.conf
       └── graph/
   ```
5. 重启游戏
6. 查看游戏日志（`logs/latest.log`）确认模型是否加载成功

**注意：**
- 自定义模型会覆盖内置模型
- 如果自定义模型加载失败，会自动回退到内置模型
- 大型模型（>500MB）可能因内存不足或格式不兼容而无法加载
- 建议优先优化触发词配置，而不是追求更大的模型

### 提高识别准确度的技巧

1. **优化触发词**：
   - 使用2-4个字的简短词汇
   - 避免生僻字和复杂词汇
   - 不同法术使用差异明显的触发词
   - 避免发音相似的词（如"护盾"和"黑洞"的首字母都是 h）

2. **优化环境**：
   - 在安静环境下使用
   - 使用质量更好的麦克风
   - 调整麦克风音量到合适水平

3. **优化说话方式**：
   - 说话清晰，发音标准
   - 语速适中，不要太快或太慢
   - 保持稳定的音量
